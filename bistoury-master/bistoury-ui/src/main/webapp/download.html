File download usage statement:

Large files may take up system resources for a long time. Please pay attention to the following points when using：

1. In order to minimize the impact of file download on the system load, bistoury limits the file download speed to 10m / s. large files may occupy system resources for a long time, and the download will be forced to stop for more than an hour.

2. Only the files in the specified directory can be downloaded. If the files to be downloaded are not in the directory provided by default, you can contact the administrator for configuration.

3. The file download function is not supported in the lower version. If you need to use this function, please upgrade the agent first
